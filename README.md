## üíª How to Use

### **Install system dependencies**

- Install [Git](https://git-scm.com).
- Install [Node.js](https://nodejs.org/en/download/).

### **Clone**

In your command line:

```bash
$ git clone https://github.com/r-e-d-ant/rundictionary.git
$ cd rundictionary

# Install dependencies
$ npm install

# Run the app
$ npm start
```
